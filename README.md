Présentation : Site Portfolio de John Doe
1. Contexte du Projet
Ce projet est réalisé dans le cadre du titre professionnel de Développeur Web et Web Mobile. L'objectif est de concevoir et de développer un site "Portfolio" vitrine mettant en avant le profil d'un professionnel du numérique, en l'occurrence, John Doe.

Ce support sert à la fois de démonstration technique des compétences acquises et d'outil de communication pour le professionnel.

2. Objectifs et Fonctionnalités
Le site est conçu pour offrir une expérience utilisateur fluide et regroupe l'ensemble des informations nécessaires à la compréhension du profil de l'expert.

Identité et Compétences
Présentation du profil : Une mise en avant du domaine d'activité et des services proposés (développement frontend, backend, conseil, etc.).

Évaluation technique : Un module dédié présentant les niveaux de compétences sur différents outils et langages de programmation.

Intégration GitHub : Un résumé dynamique ou statique du profil GitHub pour illustrer l'activité de code et les projets en cours.

Contact et Réseaux Sociaux
Pour faciliter la mise en relation, le site intègre plusieurs points d'entrée :

Coordonnées directes : Affichage clair du numéro de téléphone, de l'adresse postale et de l'adresse email.

Formulaire de contact : Une interface dédiée permettant l'envoi de messages directement depuis le site.

Écosystème social : Des liens vers les réseaux sociaux professionnels pour étendre la visibilité de John Doe.

Aspects Légaux
Conformité : Présence d'une page de mentions légales pour répondre aux obligations juridiques liées à l'édition d'un service de communication au public en ligne.
